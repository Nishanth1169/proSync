Project Management Tool
Overview
This Project Management Tool is designed to simplify project planning, task tracking, and team collaboration. It leverages Spring Boot for the backend and React for the frontend, offering a modern and efficient solution for managing projects and teams.

Features
User Authentication: Secure login and authentication system for users.
Project Management: Create, update, and track project progress.
Task Tracking: Assign tasks, set deadlines, and monitor task completion.
Team Collaboration: Foster collaboration with real-time updates, comments, and file attachments.
Dashboard: Visualize project status, task distribution, and team workload.
Notifications: Receive notifications for upcoming deadlines and task assignments.
Technologies Used
Backend:
Spring Boot
Spring Security
Hibernate ORM
RESTful APIs
Frontend:
React
Redux (optional for state management)
Material-UI (for UI components)
Database:
MySQL
